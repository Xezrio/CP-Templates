**位运算**（二进制）

​				& 	按位与	A & B		A, B均为1，结果为1; 否则为0

​				| 	 按位或	A | B		A, B任一个为1，结果为1; 否则为0

​				^ 	 按位异或	A ⊕ B		A, B不同，结果为1; 否则为0

​						A ^ B ^ C ^ D = B ^ D ^ C ^ A （顺序随意）

​						0 ^ x = x, x ^ 0 = x

​				~	  取反	~ A		取反A

​				\>>	右移	A >> n		A 除以 $${2^n}$$

​				<<	左移	A << n		A 乘 $${2^n}$$

**模运算**

​				(a + b) % p = (a % p + b % p) % p

​				(a - b) % p = (a % p - b % p) % p

​				(a * b) % p = (a % p * b % p) % p

​				 a ^ b % p = ((a % p) ^ b) % p

​				*结合律*： ((a+b) % p + c) % p = (a + (b + c) % p) % p

​								((a * b) % p * c)% p = (a * (b * c) % p) % p

​				*分配律*： ((a + b) % p * c) % p = ((a * c) % p + (b * c) % p) % p
